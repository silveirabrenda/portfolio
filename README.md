# Portfolio

Este projeto consiste em um site pessoal para apresentar minha trajetória acadêmica e profissional na área de tecnologia, sendo desenvolvido do zero, passando pelas etapas de:

- Wireframe
- Design de Interfaces (UI/UX)
- Prototipação
- Desenvolvimento
- Responsividade
- Testes e melhorias
- Deploy/Publicação
